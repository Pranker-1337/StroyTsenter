# Stroy Tsentr - Официальный сайт офиса продаж недвижимости

Современный, высокопроизводительный веб-сайт для компании **Stroy Tsentr** (Душанбе, Таджикистан). Разработан с использованием передовых технологий для обеспечения максимальной скорости загрузки, отличного пользовательского опыта (UX) и высокой конверсии.

## 🚀 Основные возможности

- **Премиальный дизайн:** Современный минималистичный интерфейс в стиле ведущих застройщиков мирового уровня.
- **Интерактивный каталог:** Удобный поиск и фильтрация объектов недвижимости по типам (квартиры, дома, коммерция) и районам.
- **Полная адаптивность:** Сайт идеально работает на всех устройствах: от смартфонов до широкоформатных мониторов.
- **Оптимизация конверсии:** Продуманные формы обратной связи, кликабельные контакты и интеграция с WhatsApp.
- **Высокая производительность:** Использование React и современных методов оптимизации обеспечивает мгновенный отклик.

## 🛠 Технологический стек

- **Frontend:** React 19
- **Стилизация:** Tailwind CSS 4 (современный движок стилей)
- **Анимации:** Motion (для плавных переходов и интерактивности)
- **Иконки:** Lucide React
- **Сборка:** Vite (максимальная скорость разработки и работы)

## 📂 Структура проекта

- `src/App.tsx` — Основная логика и структура лендинга.
- `src/constants.ts` — Централизованное управление контентом (объекты, отзывы, преимущества). Легко редактировать без изменения кода.
- `src/index.css` — Глобальные стили и конфигурация темы.

## 🔧 Установка и запуск

1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите сервер для разработки:
   ```bash
   npm run dev
   ```
3. Сборка для продакшена:
   ```bash
   npm run build
   ```

## 📈 Преимущества для бизнеса

Данный сайт спроектирован как полноценный инструмент продаж. Он заменяет оффлайн-презентации, повышает доверие к бренду и автоматизирует сбор заявок от потенциальных покупателей.
